# IMEI号如何查看？

检测流程描述：
①通用方法：打开拨号键盘，输入*#06#，查看IMEI
②苹果：打开设置-关于本机-IMEI

# IMEI号无法读取/异常应该怎么检测？

检测流程描述：
当IMEI号无法填写时，可在IMEI号一栏填写订单号即可
注意：
如无基带不显示IMEI号的情况下，可以用序列号查询保修等信息

# IMEI号怎么录入？

检测流程描述：
录入IMEI1
注意：
如无法开机/无法正常显示等无法查看IMEI的情况下可填写订单号

# 手机无IMEI号，造成无法填写怎么检测？

检测流程描述：
手机无IMEI号，无法填写，IMEI号一栏填写订单号回收即可

# imei1 正常，imei2 显示空应该怎么选？

检测流程描述：
imei1 正常，imei2 显示为空为正常，无需勾选任何问题

# 苹果手机存储容量如何查看？

检测流程描述：
①打开设置-通用-关于本机-总容量
②使用设备的IMEI/SN码到对应品牌官网/爬虫/公众号进行查询

# 手机更换过内存，应该怎么选？

检测流程描述：
勾选：【实际关于本机显示的容量】
现象详细描述：
手机更换过内存容量，按实际关于本机显示的容量勾选
注意：
①如更换官方没有的内存组合则不回收
②如查询（官网/爬虫/公众号）的出厂容量大小与实物容量大小不符，则还需勾选【主板拆修】-【外部-主板（工具提示异常、内存与出厂不符）】

# 外壳更换过，机身颜色应该怎么选？

检测流程描述：
勾选：【机身外壳实际颜色】
现象详细描述：
更换后壳的机型选更换后的颜色，以机身外壳实际颜色勾选
注意：
部分机型颜色命名与观感差距较大的需结合官网查询颜色名称勾选

# 如何手动测试光线感应功能

检测流程描述：

注意：
①部分机型亮度条变化较慢，显示亮度有变化即可
②部分机型光线传感器在屏幕下半部分，如照射上半部分无反应需要尝试照射其他位置
③注意核实自动亮度调节是否打开，质检前调整屏幕亮度至20%，环境光亮度正常的情况下不会出现自动调节到最亮或最暗，如果一直自动最高亮度或最低亮度，需要核实是否异常

注意：

# 屏幕有色块/斑块，应该怎么选？

检测流程描述：
若屏幕显示出现在任意背景色都可以看到的（黑色、紫色、绿色、红色）色块，则勾选：【屏幕显示异常】-【屏幕有漏液/条纹/闪烁/图层错乱】

若屏幕仅在显示黑色背景时有其他颜色的色块，在浅色（白色和灰色）背景时正常（如浅色背景也有就算色斑），则勾选：【屏幕显示异常】-【显示黑色背景时有色块】

若屏幕在息屏状态下观看内屏的材质有变色（一般原因为屏幕受高温灼伤或时间久了密封性下降导致内屏的材料氧化变质），则勾选：【屏幕显示异常】-【息屏看内屏材质变色】

若屏幕显示在浅色（白色和灰色）背景下时可看到有其他颜色的斑块（与漏液不同的是漏液会遮挡到显示内容，而色斑只是底色变色不会遮挡显示内容），则勾选：【屏幕显示异常】-【显示浅色背景时其他颜色的斑块】

# 屏幕有黑点/彩点，应该怎么选？

# 后壳有刻字，应该怎么选

注意：

# 后壳有刻字可以回收吗

检测流程描述：
当机身有刻字情况时

若刻字、图案包含的内容为”不文明、辱骂、反动言论、政治敏感文案、邪教宣传、隐私信息（人名、地址、电话、照片）“，则不可回收

若刻字、图案包含的内容不存在”不文明、辱骂、反动言论、政治敏感文案、邪教宣传、隐私信息（人名、地址、电话、照片）“，则勾选【外壳碎裂/刻字/脱胶】-【后盖有刻字（图）或脱胶/缝隙】，正常回收即可

若3mm＜外壳磕碰/掉漆最大直径≤6mm，且磕碰/掉漆总数量≤10处，则勾选【外壳磕碰/掉漆】-【3mm＜磕碰/掉漆≤6mm且数量≤10】

若0.5mm＜外壳磕碰/掉漆最大直径≤3mm，且磕碰/掉漆总数量≤10处，则勾选【外壳磕碰/掉漆】-【0.5mm＜磕碰/掉漆≤3mm且数量≤10】

若2mm＜外壳磕碰/掉漆最大直径≤3mm，且磕碰/掉漆总数量≤10处，则勾选【外壳磕碰/掉漆】-【0.5mm＜磕碰/掉漆≤3mm且数量≤10】

若外壳磕碰/掉漆最大直径≤0.5mm，且磕碰/掉漆总数量＜3处，则勾选【外壳磕碰/掉漆】-【无以上问题】

# 后壳有磨损，应该怎么选

检测流程描述：
当后壳有磨损情况时，需要使用相机拍照，拍照角度垂直于后盖，拍照距离为画面刚好漏出后盖四周

若拍照可以体现磨损，且磨损面积大于整个后盖的三分之一，则勾选【外壳磨损】-【后盖-磨损（可拍出）面积＞1/3】

若拍照可以体现磨损，且磨损面积小于等于整个后盖的三分之一，则勾选【外壳磨损】-【后盖-磨损（可拍出）面积≤1/3】

若拍照无法体现磨损，但肉眼可见磨损总数量（不区分磨损大小）≥3处，则勾选【外壳磨损】-【后盖-磨损（不可拍）磨损≥3处或＞3mm】

若拍照无法体现磨损，但肉眼可见磨损单处最大直径＞3mm，则勾选【外壳磨损】-【后盖-磨损（不可拍）磨损≥3处或＞3mm】

若拍照无法体现磨损，但肉眼可见磨损单处最大直径＜3mm，且磨损总数量＜3处时，则无需勾选【外壳磨损】问题，按照正常即可

# 是否需要勾选电池数据修改

检测流程描述：
若电池未更换/更换原厂电池/【电池-正品部件】，且符合下列任一条件之一时，则勾选【电池拆修情况】-【电池数据修改】
条件1:当关于本机电池健康度=100%，且充电次数＞300次
条件2:当关于本机电池健康度≤98%，且充电次数＜10次
条件3:当90%≤关于本机电池健康度≤99%，且充电次数＞1000次
条件4:当80%≤关于本机电池健康度≤89%，且充电次数＞1800次
条件5:当关于本机电池健康度≥80%，且关于本机电池健康减去一根线读取电池健康度的差值≥6%（一根线读取电池健康度＞关于本机电池健康度时，无需判定条件5）

若关于本机电池健康度＜80%时，则无需勾选【电池拆修情况】-【电池数据修改】，按照实际电池健康度勾选即可

若电池有其他拆修情况，则无需勾选【电池拆修情况】-【电池数据修改】，按照【实际最严重质检项】勾选即可

注意：
关于本机电池健康度数值为最优先

# 如何检测屏幕内屏材质？

检测流程描述：
1.出厂材质判断方法：
①参考“手机性能排行APP”或到官网、中关村网站查询配置。
②曲面屏机型和使用屏幕指纹的机型用的一定是OLED屏。
2.当前材质判断方法：
①LCD屏显示白色背景时屏幕边缘的亮度对比中间会有微微的偏亮或偏暗或有色差，OLED屏屏幕整体亮度和颜色会比较均匀。
②LCD屏显示纯黑色背景时，屏幕显示为深灰色，在暗光环境下看屏幕仍在发光，OLED屏显示纯黑色背景时，屏幕显示为纯黑色，屏幕不发光，与息屏的效果一致。
③屏幕显示有亮点/亮斑/压伤的一定是LCD屏。
